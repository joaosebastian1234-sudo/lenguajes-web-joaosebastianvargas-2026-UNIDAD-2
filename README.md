# Desarrollo de Aplicaciones Web - Unidad 2

Este repositorio contiene las actividades prácticas de la **Unidad 2**, enfocadas en el dominio de los mecanismos internos de JavaScript y la implementación de lógica de programación funcional.

## 🚀 Estructura del Proyecto
La carpeta principal de esta entrega se encuentra en:  
`unidad2/semana1/`

### Contenido de la Semana 1:
1. **`analisis.md`**: Informe técnico sobre Hoisting, Coerción de tipos y gestión de estados `null` vs `undefined`.
2. **`logica.js`**: Procesamiento de datos bibliotecarios utilizando métodos funcionales (`filter`, `map`, `reduce`, `sort`).
3. **`seguridad.js`**: Implementación de seguridad y encapsulamiento mediante **Closures**.
